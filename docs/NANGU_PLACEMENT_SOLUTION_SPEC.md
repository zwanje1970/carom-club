# 난구해결사 - 공배치 & 해법 (축약본)

## 1. 공배치 (Placement)

**목적**: 공 위치만 설정

### 기능
- **드래그 이동**: 공 선택 후 드래그로 위치 변경
- **선택 표시**: 공의 1.5~2배 반투명 원
  - 공 색상 / opacity 0.2~0.35
  - 공 위 레이어, 함께 이동
- **터치 영역**: 실제보다 1.5배 (보이지 않음) — 터치 편의
- **상단 안내**
  - 위치: 상단 중앙 고정
  - 스타일: 배경 없음 / 14~16px / bold
  - 문구: 🔴공 이동 / 🟡공 이동 / ⚪공 이동
  - 드래그 시작: 표시 / 종료: 숨김

### 제외
- ❌ 당점, 두께, 속도, 경로선
- ❌ 진동, 애니메이션

---

## 2. 해법 (Solution)

**진입**: 공배치 완료 클릭 후 (제목·설명 입력 후 게시 → 상세에서 해법 제출)

### 기능
- 당점 / 두께 / 스트로크 / 스피드 / 경로선

### 방식
- 선 직접 그림
- 스팟 포인트로 수정

---

## 3. 원칙
- 👉 **공배치 = 위치만**
- 👉 **해법 = 이후 처리**
- 👉 **완전 분리 구조**
