# Step 14: 공개 대진표·결과 관람 UI (Public Bracket and Results UI)

## 1. 개요

이번 단계는 **관리자 실무가 아닌 공개 관람용** UI입니다. 일반 사용자·참가자·관람자가 **로그인 없이** 권역 예선 대진표, 본선 대진표, 경기 결과, 진행 상태를 쉽게 볼 수 있도록 했습니다.

- **조회 전용**: 수정·결과 입력·결제 기능 없음.
- **모바일 우선**: 좁은 화면에서도 읽기 쉽게 카드형·라운드별 구분.
- **권역 예선 ↔ 본선**이 한 대회 안에서 자연스럽게 이어지도록 구성.

---

## 2. 공개 관람 라우트 구조

| 경로 | 설명 |
|------|------|
| /tournaments | 대회 목록 (기존) |
| /tournaments/[id] | 대회 상세. 진행 상태 배지 + "대진·결과 관람" 링크 (권역 예선, 본선 대진표, 경기 결과) |
| /tournaments/[id]/zones | 권역 목록 (카드형, 진행률·완료/진행중 배지) |
| /tournaments/[id]/zones/[tzId] | 권역 상세 + 권역 대진표 |
| /tournaments/[id]/bracket | 본선 대진표 (라운드별 컬럼/가로 스크롤) |
| /tournaments/[id]/results | 경기 결과 요약 (권역·본선 완료 현황, 링크) |

- **로그인 불필요**: 모든 위 경로는 세션 없이 접근 가능.
- **비공개 대회**: `status === "HIDDEN"` 인 대회는 공개 API/페이지에서 404 처리 (`getPublicTournamentOrNull`).

---

## 3. 공개 대회 상세 화면

- **기본 정보**: 대회명, 일시, 장소 (기존 유지).
- **대진·결과 관람 섹션** (DB 데이터일 때 표시):
  - 진행 상태 문구 (권역 예선 진행 중 / 본선 준비 / 본선 진행 중 / 대회 종료 등).
  - 링크: **권역 예선** → /tournaments/[id]/zones, **본선 대진표** → /tournaments/[id]/bracket (본선이 있을 때만), **경기 결과** → /tournaments/[id]/results.

---

## 4. 권역 예선 관람 UI

- **권역 목록** (/tournaments/[id]/zones):
  - 권역 카드: 권역명, 참가자 수, 완료 경기 수 / 전체 경기 수, 진행률 바.
  - 상태 배지: 완료 / 진행 중 / 대기.
  - 카드 클릭 → 권역 상세.
- **권역 상세** (/tournaments/[id]/zones/[tzId]):
  - 권역 기본 정보, 참가자 수, 경기 완료/전체.
  - **권역 대진표**: 라운드별(1라운드~결승) 경기 카드 리스트.
  - 경기 카드: A vs B, 점수, 승자 강조, 상태 배지(예정/진행중/완료/부전승).
  - **참가자명 검색**: 입력 시 해당 이름이 포함된 경기만 필터.

---

## 5. 본선 대진표 UI

- **경로**: /tournaments/[id]/bracket.
- **본선이 없을 때**: "본선 대진이 아직 생성되지 않았습니다." 메시지.
- **본선이 있을 때**:
  - 라운드별 컬럼 (1라운드, 2라운드, 준준결승, 준결승, 결승).
  - 각 경기 카드: A vs B, 점수, 승자 강조, 상태 배지.
  - **모바일**: 가로 스크롤으로 라운드 컬럼 탐색.
  - **참가자명 검색**: 해당 이름이 포함된 경기만 표시.

---

## 6. 경기 결과/상태 표시

- **상태 배지**: 예정(회색), 진행중(파란), 완료(녹색), 부전승(슬레이트).
- **표시 요소**: 참가자 A/B, 점수, 승자(굵게), 라운드명.
- 관리자용 입력 UI가 아닌 **관람용 결과** 스타일로만 노출.

---

## 7. 경기 결과 요약 페이지

- **경로**: /tournaments/[id]/results.
- **내용**: 진행 상태, 권역 예선 완료/전체 경기 수(진행률 바), 권역별 "결과 보기" 링크, 본선 완료/전체(진행률 바), "본선 대진표 보기" 링크.

---

## 8. 공개 조회 API

관리자 API와 분리된 **읽기 전용** 공개 API입니다.

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | /api/public/tournaments/[id] | 대회 요약 (이름, 주최, 진행 상태, 권역 수, 본선 여부, 참가자 수 등). 비공개(HIDDEN) 시 404. |
| GET | /api/public/tournaments/[id]/zones | 권역 목록 + 권역별 참가자 수, 경기 수, 완료 수. |
| GET | /api/public/tournaments/[id]/zones/[tzId] | 권역 상세 (참가자 수, 경기 완료/전체). |
| GET | /api/public/tournaments/[id]/zones/[tzId]/bracket | 권역 대진표 (경기 목록 + 참가자명, 점수, 승자, 상태). |
| GET | /api/public/tournaments/[id]/final-bracket | 본선 대진표 (경기 목록 + 참가자명, 점수, 승자, 상태). |
| GET | /api/public/tournaments/[id]/results | 경기 결과 요약 (권역/본선 완료 현황, 권역 목록). |

- **공통**: 로그인 불필요. `getPublicTournamentOrNull(id)`로 비공개 대회 차단.
- **민감 정보 제외**: 참가자 이름만 노출, 연락처·이메일 등 미포함.

---

## 9. 모바일 UX

- **카드형 UI**: 권역·경기 단위를 카드로 구분해 터치·가독성 확보.
- **라운드별 구분**: 본선은 라운드별 컬럼 + 가로 스크롤로 긴 대진 처리.
- **진행률 바·상태 배지**: 현재 상태를 작은 공간에 빠르게 전달.
- **참가자명 검색**: 모바일에서도 입력 가능한 한 줄 검색으로 "내 경기" 찾기 지원.

---

## 10. 참가자 찾기/필터

- **권역 대진표·본선 대진표**: "참가자명 검색" 입력 시, 해당 이름이 A 또는 B에 포함된 경기만 표시 (클라이언트 필터).
- 권역 필터·본선만/예선만·완료만 보기는 추후 확장 가능하도록 구조만 열어둠.
